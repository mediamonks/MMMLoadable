MMMBackoffTimeoutPolicy

A timeout policy that is using a constant period after successful attempts and a set of incrementing timeouts after failures.

  • init(period:min:max:multiplier:)
  • reset()
  • nextTimeout(afterFailure:)
  • period

    How often to sync after a success. Set to 0 to disable.

    You can change this but note that it will be used only on the next change in the state of the loadable, because there is no feedback between the policy and the syncer.
